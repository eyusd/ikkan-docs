## Installation

To get started with **Ikkan**, you need to install the package from your favorite package manager:
<Tabs items={['pnpm', 'npm', 'yarn', 'bun']}>
  <Tab value="pnpm">
    <DynamicCodeBlock lang="bash" code="pnpm dlx ikkan@latest init" />
  </Tab>
  <Tab value="npm">
    <DynamicCodeBlock lang="bash" code="npx ikkan@latest init" />
  </Tab>
  <Tab value="yarn">
    <DynamicCodeBlock lang="bash" code="yarn dlx ikkan@latest init" />
  </Tab>
  <Tab value="bun">
    <DynamicCodeBlock lang="bash" code="bunx --bun ikkan@latest init" />
  </Tab>
</Tabs>
